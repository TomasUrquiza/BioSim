# BioSim
BIOSIM: Universe Evolution Engine es un simulador avanzado que integra principios biológicos y el modelo matemático Lotka-Volterra en un entorno "Neon-Cyberpunk". Con un universo de $3000 \times 3000$ unidades, permite observar el equilibrio entre presas y depredadores mediante metabolismos de energía, reproducción y cámaras con navegación libre.
